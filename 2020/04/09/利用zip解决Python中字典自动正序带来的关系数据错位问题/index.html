本问题最初出现在构建蛋白质的关系字典中

  • 由于在字典中，数据的排序是自动正序的（也可以理解为乱序），在一组关系数据列表中，
    我们构建字典的方式往往是先提取一组数据构建字典的键，再提取另一组数据一一对应为键的值 。但是由于字典的键是乱序的，那么对应的值也必然是错误的。本方法就是为了解决这个问题。
